# Skillcruit - IA de Recrutement Prédictive

Skillcruit est une plateforme innovante qui utilise l'Intelligence Artificielle pour automatiser et optimiser la présélection des candidats. Notre solution permet de réduire les biais cognitifs tout en libérant les recruteurs des tâches répétitives.

## 🚀 Fonctionnalités Clés

- **Scoring Intelligent** : Évaluation objective des CV basée sur le NLP (Natural Language Processing).
- **Matching Dynamique** : Identification instantanée des meilleurs profils pour chaque offre.
- **Résumés Automatiques** : Synthèse automatique du parcours professionnel des candidats.
- **Conformité RGPD** : Gestion sécurisée des données personnelles.

## 🛠 Technologies Utilisées

- **Frontend** : React, Vite, TypeScript, Tailwind CSS
- **UI Components** : shadcn/ui, Framer Motion
- **Data Visualization** : Recharts
- **Forms & Validation** : React Hook Form, Zod

## 💻 Installation Locale

1. Clonez le dépôt :
   ```sh
   git clone <REPO_URL>
   ```

2. Installez les dépendances :
   ```sh
   npm install
   ```

3. Configurez les variables d'environnement (EmailJS) :
   ```sh
   cp .env.example .env
   ```

4. Lancez le serveur de développement :
   ```sh
   npm run dev
   ```
